# Atividade: Lógica de Programação e Algoritmos 🚀
## Curso de Desenvolvimento de Sistemas - SENAI

Este repositório contém uma série de exercícios práticos desenvolvidos para a disciplina de Lógica de Programação. O objetivo principal foi exercitar a construção de algoritmos através de **pseudocódigos** e a visualização lógica por meio de **fluxogramas**.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **[VisuAlg](https://visualg3.com.br/):** Utilizado para o desenvolvimento e execução dos pseudocódigos (`.alg`).
* **[Draw.io](https://app.diagrams.net/):** Ferramenta utilizada para a criação dos diagramas de fluxo (fluxogramas).
* **GitHub:** Para versionamento e organização dos arquivos.

---

As atividades cobrem cálculos matemáticos, estruturas condicionais e lógica essencial. Abaixo estão os exercícios realizados:

## 🚀 Como executar os arquivos

1.  **Pseudocódigos (`.alg`):**
    * Baixe e instale o **VisuAlg**.
    * Abra o programa e carregue o arquivo desejado.
    * Pressione `F9` para executar o algoritmo.

2.  **Diagramas:**
    * Os arquivos em `.png` ou `.pdf` podem ser visualizados diretamente no navegador.
    * Os arquivos com extensão `.drawio` podem ser editados importando-os no site [diagrams.net](https://app.diagrams.net/).
